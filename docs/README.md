# เอกสารโปรเจกต์ MediQueue

โฟลเดอร์นี้เก็บเอกสารอธิบายภาพรวม การใช้งาน และการติดตั้งระบบ

| ไฟล์ | รายละเอียด |
|------|-------------|
| **01_PROJECT_OVERVIEW.md** | สรุปภาพรวมโปรเจกต์ เทคโนโลยี โครงสร้าง บทบาท ฟีเจอร์หลัก โมเดลข้อมูล และความปลอดภัย |
| **02_USER_GUIDE_BY_ROLE.md** | ขั้นตอนการใช้งานแยกตาม Role: ผู้ป่วย (จอง/เช็คอิน/แดชบอร์ด), แอดมิน (คิว/จอง/ตั้งค่า), แพทย์ (เรียกคิว/หมายเหตุ/จบการตรวจ) |
| **03_INSTALLATION_AND_DATABASE.md** | การติดตั้งลงเครื่อง (Node, MongoDB), การตั้งค่า .env, การรัน Backend/Frontend, การเตรียมฐานข้อมูลและรัน seed |

แนะนำให้อ่านตามลำดับ 01 → 02 → 03 หากเริ่มจากศูนย์
