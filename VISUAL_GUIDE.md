## 📊 Color Scheme

```
PRIMARY COLORS:
🔵 Blue (Main): #1E88E5
   - Headers, buttons, links
   - Active states
   
🟢 Green (Success): #2E7D32, #4CAF50
   - Confirmed bookings
   - Completed actions
   
🟡 Orange (Warning): #FBC02D
   - Nearly full queues
   - Important alerts

🔴 Red (Error): #D32F2F
   - Cancellations
   - Urgent notices

⚫ Gray (Neutral): #364153, #4a5565
   - Text, secondary info
   - Borders, dividers
```
